# 2-后端

放各类后端服务：控制面板后端、APK 服务后端、母体后端、sub2api 后端、公共接口。

## 这个目录负责什么
- 承载可部署的服务进程。
- 把前端、设备端、母体端、调度端解耦。
- 统一 API、DTO、事件、鉴权和健康检查。

## 子目录
- 1-控制面板后端：总览、管理、配置、健康检查、运维接口。
- 2-APK服务后端：Android / MiClaw / 设备同步 / 设备控制。
- 3-母体后端：母体主后端服务、Gateway 入口、主业务 API。
- 4-sub2api后端：模型调度、token 管理、provider 路由、统计。
- 5-公共接口：跨模块共享接口、协议定义、事件定义、DTO。
