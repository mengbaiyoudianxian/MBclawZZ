### 架构分层

- **Stage A — 存储层**（8项目 ✅）：用户/项目/会话CRUD、消息、分类树、关键词、分层搜索(L1关键词→L2 TF-IDF→L3 ChromaDB)、工具索引、模型注册

- **F系列 — 反馈与画像**（2项目 ✅）：主动反馈、用户心理画像
- **P系列 — 扩展**（2项目 ✅）：思维碰撞引擎、乌托邦计划

## 四、参考素材项目

### miclaw-apk-analysis（MiClaw APK 逆向分析）

**定位**：小米 MiClaw AI 助手劫持/替换方案研究

**核心发现**：
- MiClaw 拥有 **386 个工具**，覆盖 **18 个分类**（WiFi/蓝牙/短信/通话/文件/相机/录屏/日历/笔记/浏览器/地图/购物等）
- 支持 **MCP 协议**
- 语音唤醒（AIVSE SDK）
- 当前AI内核：小米 `api.mify.mioffice.cn`

**三种攻击方案**：
- 方案A（Hosts劫持+本地代理）：修改hosts → Flask HTTPS MITM → 转发至OpenClaw Gateway
- 方案B（APK反编译+重打包）：apktool解包 → 替换API URL → 重打包 + core-patch签名绕过
- 方案C（MCP Server注入）：注册自定义MCP Server拦截工具调用

**代码文件**：
- `hybrid_engine/deepseek_brain.py` — DeepSeek AI混合分析引擎
- `proxy/miclaw_proxy.py` — LLM代理（Flask, OpenAI格式转发至OpenClaw）
- `proxy/setup.sh` — Android设备设置脚本

## 七、技术亮点

1. **双态记忆存储**：snapshot（稳定快照）+ live（活跃会话），批量原子操作
2. **Hermes 6模式**：声明式/过程式分离，MEMORY.md + User.md 持久记忆
3. **三层语义搜索**：L1关键词(<10ms) → L2 TF-IDF(<100ms) → L3 ChromaDB向量(<500ms)
4. **Curator自动归档**：30天stale / 90天archived，自动整理
5. **写入审批门**：风险评分 + 用户可设阈值，安全可控
6. **思维碰撞引擎**：组合创新，多方案并行探索
7. **11平台统一网关**：一套代码对接所有主流IM平台
