# MiClaw 实例架构详解

## 什么是 MiClaw 实例

MiClaw 是小米的 AI 云端服务 (xiaomi/mimo-pro 模型)。MBclaw 通过桥接让用户"白嫖"这个算力。用户不需要自己的 API Key。

## 三层架构

```
┌─────────────────────────────────────────────────────────┐
│                        APK 手机                          │
│  用户点击"白嫖算力" → 自动配置后端地址                    │
│  所有 LLM 请求走: /bridge/miclaw/v1/chat/completions     │
└─────────────────────┬───────────────────────────────────┘
                      │ HTTP
                      ▼
┌─────────────────────────────────────────────────────────┐
│              层1: 存储机 47.83.2.188                      │
│  ┌───────────────────────────────────────────────────┐  │
│  │ nginx                                             │  │
│  │ /bridge/miclaw/v1/ → proxy_pass 100.126.55.0:8765 │  │
│  └───────────────────────────────────────────────────┘  │
│  ┌───────────────────────────────────────────────────┐  │
│  │ router.py + bridge_manager.py                     │  │
│  │ • POST /bridge/miclaw/apply       创建实例         │  │
│  │ • POST /bridge/miclaw/login/{id}  验证Mi凭证       │  │
│  │ • GET  /bridge/miclaw/status      查询状态         │  │
│  │ • 实例数据: /var/lib/mbclaw/miclaw_instances.json  │  │
│  │ • 超时清理: 2小时未登录自动销毁                     │  │
│  └───────────────────────────────────────────────────┘  │
└─────────────────────┬───────────────────────────────────┘
                      │ Tailscale 100.126.55.0:8765
                      ▼
┌─────────────────────────────────────────────────────────┐
│            层2: 工具池 121.199.57.195                     │
│  ┌───────────────────────────────────────────────────┐  │
│  │ miclaw_api_bridge (Rust binary)                    │  │
│  │ PID 430032 | 监听 0.0.0.0:8765                    │  │
│  │ 路径: /root/miclaw_api_bridge/src-tauri/target/   │  │
│  │       release/miclaw_api_bridge                    │  │
│  │ GitHub: github.com/NEORUAA/miclaw_api_bridge       │  │
│  └───────────────────────────────────────────────────┘  │
└─────────────────────┬───────────────────────────────────┘
                      │ HTTP
                      ▼
┌─────────────────────────────────────────────────────────┐
│                小米 MiClaw 云端                           │
│  模型: xiaomi/mimo-pro                                   │
│  验证方式: 小米账号 + 密码                                │
│  通过 bridge 登录后获取 token 进行 API 调用               │
└─────────────────────────────────────────────────────────┘
```

## 关键源码位置

| 文件 | 服务器 | 作用 |
|------|--------|------|
| `bridge_manager.py` | 存储机 /opt/mbclaw/app/admin/ | 实例管理 + 清理 + API代理 |
| `router.py:api_miclaw_instances` | 存储机 /opt/mbclaw/app/admin/ | 管理面板展示实例 |
| `nginx mbclaw.conf` | 存储机 /etc/nginx/conf.d/ | 路由 /bridge/miclaw/v1 → 100.126.55.0:8765 |
| `miclaw_api_bridge` | 工具池 /root/miclaw_api_bridge/ | Rust桥接到小米API |
| `MiclawBridge.kt` | APK源码 | 手机端申请实例逻辑 |

## 超时清理规则

- `_cleanup_inst()` 每 2 小时检查
- 触发条件: 实例 status != "ready" 且创建时间 > 7200 秒
- 触发时机: MiClaw 页面打开时、申请实例时、查询状态时
