### k2 工作原理 (cc_loop.py)
```
1. 读 app/encoder.py (当前代码)
2. 构造 prompt: "Read X. Current code: [...] Task: add l1_judge()..."
3. 调 Claude (DeepSeek Anthropic端点) 生成新代码
4. 跑 pytest: 必须 >=50 passed 且 0 failed
5. 调 Qwen 评审: 评分 >=0.7 才接受
6. 备份→替换→再测试→通过则保留
7. 循环 5 个文件轮流转
```
