# Auto Mode（全自动决策模式）— 永久移出 Core

## 原设计核心
- 文件：`auto_mode.py` (103 行)
- 思路：LLM 自评方案 → 自动选最优 → 自动执行；用户不响应则自研究多方案让用户选

## 否决理由
1. **安全边界未定义就做"自主决策"**——工程伦理红线
2. R0 无 Agent，无"自动执行"载体
3. 多方案并行 = N 倍 API 成本，无验证收益
4. "用户没响应"判定阈值无依据

## 重启条件
- Agent Runtime R2 真实上线后
- 有具体场景（如"代我归档 N 条 stale 经验"）
- 明确白名单 + 显式审批
