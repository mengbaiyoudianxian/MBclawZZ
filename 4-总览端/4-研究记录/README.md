# 研究记录

## 当前重点
1. `mbclaw-mother`：母体主线，包含 gateway、memory、tokenpool、tools、runtime。
2. `OpenClaw`：可借鉴的平台化、skills、插件与执行层。
3. `LiteLLM`：模型路由与兼容层。
4. `mem0` / `GraphRAG`：记忆与知识组织。
5. `OpenHands` / `Claude-Code` / `Codex-CLI`：agent 与开发工作流。
6. `agentskills`：技能规范。

## 研究原则
- 先看 README / 规范，再看实现。
- 先抓职责边界，再看内部细节。
- 先做可复用总结，再考虑移植。
